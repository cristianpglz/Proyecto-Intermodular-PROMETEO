# Proyecto Intermodular 1º SMR - Gimnasio Prometeo

**Sistema de Gestión Integral para un centro deportivo de alto rendimiento**

Proyecto completo de 1º de Técnico en Sistemas Microinformáticos y Redes (Prometeo by The Power).

## 📋 Descripción
Gimnasio Prometeo es un centro deportivo de 850 m² con más de 450 socios. Este proyecto diseña desde cero toda su infraestructura tecnológica: red segmentada con VLANs e IoT, sistemas operativos, hardware, documentación ofimática y base de datos.
