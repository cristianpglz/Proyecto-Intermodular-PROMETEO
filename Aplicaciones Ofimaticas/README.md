# Aplicaciones Ofimáticas (0223) - Gimnasio Prometeo

**Documentación ofimática completa del Proyecto Intermodular**

En esta carpeta se encuentran todos los documentos Word y Excel generados para el módulo de **Aplicaciones Ofimáticas**.

## 📄 Archivos incluidos

| Archivo                        | Descripción |
|--------------------------------|-----------|
| `Descripcion_Estructura.docx`  | Documento descriptivo oficial de la empresa (portada, índice, descripción, empleados e infraestructura) |
| `Montaje_de_equipos.docx`      | Documento completo de montaje y mantenimiento de equipos (duplicado del módulo 0221) |
| `Inventario.xlsx`              | Inventario completo de hardware y software (duplicado del módulo 0221) |
| `Registro_Incidencias.xlsx`    | Registro de incidencias técnicas con formato condicional y fórmulas |
| `SO intermodular.docx`         | Documento de apoyo con información del sistema operativo |

## ℹ️ Nota importante

Los archivos **Montaje_de_equipos.docx** e **Inventario.xlsx** están **duplicados** en esta carpeta para facilitar la revisión.  
De esta forma, el módulo de Aplicaciones Ofimáticas contiene **todos los documentos necesarios en un solo lugar**, sin necesidad de saltar entre carpetas.
