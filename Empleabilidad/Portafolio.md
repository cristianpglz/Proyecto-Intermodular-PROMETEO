# Portfolio Profesional: Proyecto Gimnasio Prometeo

Este portfolio resume la implementación técnica integral de un sistema de gestión para un centro deportivo, abarcando desde la infraestructura física hasta la lógica de datos.

## 1. Infraestructura de Sistemas y Hardware
En esta fase me encargué de asegurar que el entorno físico y digital fuera el adecuado para el negocio.

* **Montaje de Equipos**: Selección y ensamblaje de componentes para los puestos de administración, optimizando la relación rendimiento/precio.
* **Sistemas Operativos**: Instalación y securización de Windows Server para la base de datos y puestos de trabajo con políticas de usuario restrictivas.
* **Ofimática Avanzada**: Creación de un sistema de facturación y control de inventario mediante hojas de cálculo automatizadas.

## 2. Diseño de Redes y Comunicaciones
Diseño de una arquitectura de red segura para dar soporte a todos los dispositivos del centro.

* **Segmentación de Red**: Configuración de una red local (LAN) con VLANs para separar el tráfico administrativo del acceso Wi-Fi de clientes.
* **Seguridad**: Implementación de reglas de firewall para proteger el acceso a los datos de los socios.

## 3. Arquitectura de Datos (Backend)
El corazón lógico del proyecto, diseñado para una gestión sin errores.

* **Modelo Entidad-Relación**: Diseño de una base de datos robusta que gestiona socios, pagos, actividades y empleados.
* **Lógica de Reservas**: Implementación de relaciones complejas para permitir que múltiples socios se inscriban en diversas actividades sin solapamientos.

## 4. Conclusión y Aprendizajes
* **Visión Global**: He aprendido que un proyecto tecnológico no es solo código; requiere una base sólida de hardware y una red segura para funcionar.
* **Perfil Híbrido**: Este proyecto demuestra mi capacidad para gestionar sistemas físicos mientras me especializo en el desarrollo de estructuras de software.
