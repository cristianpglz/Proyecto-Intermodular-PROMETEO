# Investigación del Sector Profesional

### 1. Empresas del Sector (Enfoque Web y Sistemas)

| Empresa | Qué hace | Perfiles que contrata | Tecnologías |
| :--- | :--- | :--- | :--- |
| **Idealista** | Portal inmobiliario líder. Su negocio es 100% digital y basado en datos. | Desarrolladores Frontend y Backend, ingenieros de datos y QA. | Java, Spring Boot, JavaScript (React/Vue) y SQL. |
| **CTAG** (O Porriño) | Centro tecnológico de automoción. Desarrollan interfaces (HMI) y sistemas de conectividad. | Desarrolladores de interfaces, ingenieros de software y técnicos de sistemas. | HTML5/CSS3, JavaScript, Node.js y Python. |

### 2. Perfiles profesionales a seguir

* **Perfil 1: Frontend Developer**
    * **Qué hace**: Traduce diseños en código interactivo para el navegador.
    * **Habilidades**: JavaScript, CSS dinámico y frameworks como React.
* **Perfil 2: Full-Stack Developer**
    * **Qué hace**: Trabaja tanto en la base de datos como en la interfaz visual.
    * **Habilidades**: Node.js, SQL y control de versiones con Git.
