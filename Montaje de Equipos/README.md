# Montaje y Mantenimiento de Equipos (0221) - Gimnasio Prometeo

**Documentación completa de hardware y montaje del Proyecto Intermodular**

En esta carpeta se encuentra toda la documentación relativa al módulo de **Montaje y Mantenimiento de Equipos**.

## 📄 Archivos incluidos

| Archivo                        | Descripción |
|--------------------------------|-----------|
| `Inventario.xlsx`              | Inventario completo de equipos informáticos 
| `Registro_Incidencias.xlsx`    | Registro completo de las incidencias 
(hardware y software) |
| `Presentacion GIMNASIO.pdf`    | Presentación en PDF del sistema informático del gimnasio |
| `Montaje_de_equipos.docx`      | Documento detallado de análisis, configuración, montaje y mantenimiento (duplicado en Aplicaciones Ofimáticas) |

## ℹ️ Nota importante

El archivo **Inventario.xlsx** y el documento **Montaje_de_equipos.docx** están **duplicados** en la carpeta `Aplicaciones Ofimaticas` para facilitar la revisión del módulo de Ofimática.  
De esta forma, ambos módulos tienen toda la información centralizada y accesible sin necesidad de cambiar de carpeta.
